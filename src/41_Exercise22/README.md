你能像教程一开始那样，通过定义相应的材质来模拟现实世界的物体吗？注意材质表格中的环境光值与漫反射值不一样，它们没有考虑光照的强度。要想正确地设置它们的值，你需要将所有的光照强度都设置为vec3(1.0)，这样才能得到一致的输出：青色塑料(Cyan Plastic)容器。
